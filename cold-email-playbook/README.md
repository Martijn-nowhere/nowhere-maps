# Cold Email Outreach Setup — Client Playbook

A complete system for building and running cold email campaigns. Built and tested for School of Recycling (June 2026).

---

## What This Is

A fully automated cold email outreach system that:
- Builds verified lead lists via Apollo
- Sends personalized 3-email sequences via Instantly
- Warms up sending mailboxes to protect deliverability
- Routes warm replies to your main inbox

## Cost Per Client (Monthly)

| Tool | Cost |
